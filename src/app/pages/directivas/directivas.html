Ejemplos de Directivas y Pipes en Angular

Directivas de Angular.

Las directivas son uno de los conceptos fundamentales de Angular y permiten extender el comportamiento del DOM. Gracias a ellas, Angular puede modificar la estructura de la vista, cambiar la apariencia de los elementos o añadir lógica personalizada directamente en el template. En Angular existen distintos tipos de directivas: las directivas Componente, las directivas Estructurales y las directivas Atributo.

Directivas de componente.

Las directivas de componente son el tipo de directiva más común en Angular. Cada componente es en realidad una directiva que define una vista, su lógica y su comportamiento. Se utilizan mediante un selector personalizado y permiten reutilizar partes de la interfaz.

Directivas atributo.

Las directivas atributo modifican la apariencia o el comportamiento de un elemento existente sin cambiar su estructura. Se aplican como atributos y suelen utilizarse para estilos, estados o enlaces de datos.

Pipes.

Las pipes en Angular permiten transformar datos directamente en la vista sin modificar el valor original. Son muy útiles para dar formato a textos, fechas, números o para depurar información de forma sencilla y declarativa.

uppercase

Transforma texto a mayúsculas

{{ nombre | uppercase }}

date

Formateo de fechas

{{ fechaActual | date:'dd/MM/yyyy' }}

currency

Formato de valores monetarios

{{ precio | currency:'EUR' }}

slice

Recorte de strings

{{ textoLargo | slice:0:20 }}...

json

Visualización de objetos (debug)

{{ usuario | json }}
